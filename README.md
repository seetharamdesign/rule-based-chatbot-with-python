# Rule-based Chatbot

A simple Python chatbot that uses keyword-based rules to answer basic questions. The bot supports greetings, motivational replies, and a small set of predefined responses without external APIs.

## Features

- Rule-based conversational responses
- Simple keyword matching for user input
- Greeting based on the current time of day
- Minimal setup with a single Python script

## Requirements

- Python 3.x

## Usage

1. Open a terminal in the project folder.
2. Run:

   ```bash
   python main.py
   ```

3. Enter your name and ask a basic question.
4. Type `bye` to exit the chatbot.

## Example Questions

- `hello`
- `how are you`
- `who are you`
- `motivate me`
- `happy`
- `what is a function`

## Notes

This project is ideal for learning how rule-based chatbots work and can be extended with more patterns and smarter response handling.
